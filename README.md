# cls-sch-pipe

Create `/lib` folder and place `ojdbcX.jar` there to make sure the oracle library is present.

## Environmental Configuration

- `ORACLE_HOST`
- `ORACLE_PORT`
- `ORACLE_SID`
- `ORACLE_USER`
- `ORACLE_PASS`

- `POSTGRES_HOST`
- `POSTGRES_PORT`
- `POSTGRES_SID`
- `POSTGRES_USER`
- `POSTGRES_PASS`
